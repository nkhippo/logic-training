# 思考トレーニングについて

このトレーニングでは、断片的で曖昧な状況の中から「何を問うべきか」を自分で定義し、
仮説を積み重ねながら判断・行動を導く力を鍛えます。

## 6つの思考タイプ

| タイプ | 何をするか |
|---|---|
| ①状況を整理する | 事実・不確かな情報・矛盾を区別する |
| ②原因を掘り下げる | 表面から根本原因まで掘り下げる |
| ③先を読む | 起きうることを複数のシナリオで見積もる |
| ④選択肢を比べる | 基準を持って最善を選ぶ |
| ⑤相手を動かす | 伝え方・順序を設計する |
| ⑥問いを疑う | 解こうとしている問い自体を確認する |

## 4つのレベル

- **レベル1**：設問に答えながら各タイプの使い方を体感する
- **レベル2**：タイプは指定される。何を確認するかは自分で決める
- **レベル3**：どのタイプをどの順序で使うかを自分で選ぶ
- **レベル4**：断片的な状況だけが与えられ、問いを立てることから始める

## 点数について

回答のたびに内部で採点を行いますが、点数は表示されません。
AIが「上司」として、答えを教えずに問い返しを行います。

## 振り返りについて

最終フィードバックの後に、思考プロセスを振り返る問いかけが届きます。
回答は任意です。2回まで回答できます。
