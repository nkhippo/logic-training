## 3. タブ別仕様

### 3-1. 穴埋めタブ（fill）

#### 問題生成仕様

| 難易度 | 文章量 | 穴抜き数 | ヒント |
|---|---|---|---|
| 1（入門） | 400〜500字 | 2個 | あり |
| 2（基礎） | 400〜500字 | 3個 | 軽いヒント |
| 3（標準） | 400〜500字 | 3個 | なし |
| 4（上級） | 400〜500字 | 4個 | なし |
| 5（超難問） | 400〜500字 | 5個 | なし |

#### 採点フィードバック仕様
- 完全補完文（全接続詞を補完した版）を表示
- 各設問に「日常例文」と「代替接続詞を使った場合の論旨変化」を表示

#### GASシート：fill

| カラム | 内容 |
|---|---|
| id | タイムスタンプ（Unix ms） |
| theme | テーマ文字列 |
| diff | 難易度（1〜5） |
| date | 保存日時 |
| text | 問題文（空欄あり） |
| answers | 正解接続詞リスト（JSON） |
| hints | ヒントリスト（JSON） |
| feedback | 採点フィードバック（JSON） |
| userAnswers | ユーザー回答（JSON） |
| lang | 言語（ja/en） |

---

### 3-2. 要約タブ（summary）

#### 問題生成仕様

| 難易度 | 文字数上限 | 圧縮率目安 | 設問数 |
|---|---|---|---|
| 1（基礎） | 300字 | 約60% | 1問 |
| 2（標準） | 400字 | 約50% | 2問 |
| 3（応用） | 500字 | 約40% | 3問 |
| 4（上級） | 500〜2000字 | 約30% | 3問 |
| 5（超難問） | 500〜4000字 | 約20% | 3問 |

#### 設問タイプ

| タイプ | 内容 | 出題難易度 |
|---|---|---|
| 主張要約 | 筆者の主張を指定文字数でまとめる | 全難易度 |
| 語句説明 | 本文中の語句の意味を説明する | 難易度2〜5 |
| 理由説明 | 筆者がそう主張する理由を説明する | 難易度3〜5 |

#### 回答方法
- テキスト入力（デフォルト）
- 写真提出（JPEG/PNG/HEIC、最大2枚）

#### GASシート：summary

| カラム | 内容 |
|---|---|
| id | タイムスタンプ |
| theme | テーマ |
| diff | 難易度 |
| date | 保存日時 |
| text | 問題文 |
| questions | 設問リスト（JSON） |
| ratio | 圧縮率 |
| lang | 言語 |

### 3-3. 批判読みタブ（critique）

#### 問題形式

| 難易度 | 形式 | 設問数 |
|---|---|---|
| 1（入門） | B形式：短い論証3問（独立） | 3問 |
| 2（基礎） | B形式：短い論証3問（独立） | 3問 |
| 3（標準） | A形式：論説文1本 | 4問 |
| 4（上級） | A形式：論説文1本 | 5問 |
| 5（超難問） | A形式：論説文1本 | 5問 |

#### 設問タイプ

| タイプ | 出題難易度 | ビジネス向け |
|---|---|---|
| 本当にそう言える？の指摘 | 全難易度 | ✅ |
| 結論が成立するための条件 | 全難易度 | ✅ |
| 反対意見への応答 | 難易度2〜5 | ✅ |
| 話の流れの整理 | 難易度3〜5 | ✅ |
| 立場が異なる人からの疑問 | 難易度4〜5 | ✅（ビジネスのみ） |
| 対偶の検証 | 難易度4〜5 | ❌（受験モードのみ） |

> ビジネスフェーズでは「対偶の検証」は出題しない。難易度4〜5には「立場が異なる人からの疑問」を出題する。

#### GASシート：critique

| カラム | 内容 |
|---|---|
| id | タイムスタンプ |
| theme | テーマ |
| diff | 難易度 |
| date | 保存日時 |
| text | 問題文（A形式のみ） |
| questions | 設問リスト（JSON） |
| feedback | 採点フィードバック（JSON） |
| form | 問題形式（A/B） |
| lang | 言語 |

## 4. バックエンド仕様（GAS）

### 4-1. デプロイ設定

```
実行ユーザー: Me（スプレッドシートオーナー）
アクセス権限: Anyone（認証なし）
バージョン: v3（gas-script-v3.js）
```

### 4-2. エンドポイント

| メソッド | パラメータ | 内容 |
|---|---|---|
| GET | `?ping=1` | バージョン確認 |
| GET | `?sheet=fill` | 穴埋め過去問一覧取得 |
| GET | `?sheet=summary` | 要約過去問一覧取得 |
| GET | `?sheet=critique` | 批判読み過去問一覧取得 |
| GET | `?sheet=ame` | 空雨傘過去問一覧取得 |
| POST | `{ action: 'delete', sheet, id }` | 過去問削除 |
| POST | `{ sheet: 'fill', ... }` | 穴埋め保存 |
| POST | `{ sheet: 'summary', ... }` | 要約保存 |
| POST | `{ sheet: 'critique', ... }` | 批判読み保存 |
| POST | `{ sheet: 'ame', ... }` | 空雨傘保存 |
